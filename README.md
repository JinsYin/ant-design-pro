# Ant Design Pro (simple)

## 创建项目

```bash
yarn create umi ant-design-pro

# > ant-design-pro
# > TypeScript
# > simple
```

亦可直接下载源码：https://github.com/ant-design/ant-design-pro/releases/tag/v5.0.0

## 准备环境

```bash
yarn
```

## 脚本

### 启动项目

```bash
npm start
```

### 构建项目

```bash
npm run build
```

### 检查代码规范

检查：

```bash
npm run lint
```

修复：

```bash
npm run lint:fix
```

### 测试代码

```bash
npm test
```
